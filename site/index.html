What it does

The tool lists conversations from your ChatGPT account, lets you search and sort them inside a terminal interface, and applies bulk archive or delete actions after confirmation.

Why it exists

ChatGPT’s web UI makes large cleanups slow. This project moves the repetitive work into a faster TUI while still authenticating through the same browser-backed flow the site expects.

Where it runs

Current support targets macOS and Linux on amd64 and arm64, with a Chrome-compatible browser available for the login step.

Core workflow

Browser login first, batch actions second.

01

Authenticate in a real browser

Run chatgpt-bulk login and complete sign-in or any challenge in Chrome, Edge, Brave, or Chromium.

02

Open the TUI

Launch chatgpt-bulk to load your conversations, paginate through them, and filter the list by title.

03

Review and apply

Multi-select entries, choose archive or delete, and confirm once instead of clicking one conversation at a time.

Install and use

Start with a release binary or build from source.

Install

# Download a release for your platform
# https://github.com/arush-sal/bulk-delete-chatgpt-conversations/releases

# Verify the binary
./chatgpt-bulk --version

Quick start

chatgpt-bulk login
chatgpt-bulk

Build from source

git clone https://github.com/arush-sal/bulk-delete-chatgpt-conversations.git
cd bulk-delete-chatgpt-conversations
make build
./chatgpt-bulk --version
