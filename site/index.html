Go TUI for ChatGPT cleanup

Bulk delete or archive ChatGPT conversations from the terminal.

This project is built for one narrow job: clean up a large ChatGPT history faster than the web sidebar allows. It authenticates through a real browser session, then gives you a focused terminal UI for filtering, selecting, and acting on conversations in batches.

Download release Read full README

Why this exists

ChatGPT cleanup is tedious in the browser. This tool reduces that to login, filter, select, confirm.

What it handles

  • bulk archive
  • bulk delete
  • title filtering and sorting
  • permanent or session-only auth

Requirements

macOS or Linux, a Chrome-compatible browser, and a valid ChatGPT login.

Install and first run

Three commands, then clean up your history.

1. Get the binary

# Download the release for your platform
# https://github.com/arush-sal/bulk-delete-chatgpt-conversations/releases

./chatgpt-bulk --version

2. Authenticate

chatgpt-bulk login

Choose permanent auth if you want reuse across launches, or session-only if you want auth kept in memory for the current run.

3. Open the TUI

chatgpt-bulk

Filter by title, multi-select conversations, then archive or delete them with a single confirmation step.

User journey

Built around the actual cleanup flow.

01

Browser-assisted auth

The tool opens Chrome, Edge, Brave, or Chromium so login and any anti-bot or MFA challenge happen in a normal browser context.

02

Conversation triage in the terminal

Load conversations, move quickly with keyboard controls, search by title, and sort before selecting a batch.

03

Bulk action with confirmation

Archive to hide chats without removing them, or delete to make them no longer visible. The action is always confirmed before it runs.

Why browser login matters

This avoids the fragile plain-HTTP approach that often runs into `403` responses or bot-protection issues when trying to automate ChatGPT directly.

Auth behavior

The app supports saved auth for repeat use and a short-lived in-memory path for one-off cleanup sessions. No OpenAI API key is involved.
